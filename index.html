3. Span

The span is a container for inline content

<span>This is a span element</span>

4. Paragraph

This is Paragraph Tag, A paragraph always starts on a new line, and is usually a block of text. 

<p>This is a paragraph element</p>

5. Pre Tag

The pre tag is used to display preformatted text in a fixed-width font.

<pre>This is a paragraph element</pre>

6. Text Formatting

HTML contains several elements for defining text with a special meaning.

Bold Tag

The HTML <b> element defines bold text, without any extra importance.

<b>This is bold text</b>

Strong Tag

The HTML <strong> element defines text with strong importance. The content inside is typically displayed in bold.

<strong>This is strong text</strong>

Italic Tag

The HTML <i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

<i>This is italic text</i>

Underline Tag

The HTML <u> element defines a part of text in underlined style.

<u>This is underlined text</u>

EM Tag

The HTML <em> element defines a part of text in italic style. The content inside is typically displayed in italic.

<em>This is emphasized text</em>

Small Tag

The HTML <small> element defines smaller text. The content inside is typically displayed in smaller font.

<small>This is small text</small>

Mark Tag

The HTML <mark> element defines highlighted text. The content inside is typically displayed in a highlighted color.

<mark>This is marked text</mark>

Del Tag

The HTML <del> element defines deleted text. The content inside is typically displayed in strikethrough.

<del>This is deleted text</del>

Ins Tag

The HTML <ins> element defines inserted text. The content inside is typically displayed in underline.

<ins>This is inserted text</ins>

Subscript Tag

The HTML <sub> element defines subscripted text.

<sub>This is subscripted text</sub>

Superscript Tag

The HTML <sup> element defines superscripted text.

<sup>This is superscripted text</sup>

7. List

HTML lists allow web developers to group a set of related items in lists.

Unordered List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default

                    
<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

Ordered List

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers (1, 2, 3, etc.) by default


<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

9. Media Tags

HTML provides several tags for embedding multimedia content, such as images, videos, and audio.

Audio Tag

The HTML <audio> element embeds audio content such as MP3, WAV, and Ogg files.

            
<audio controls>
<source src="audio.mp3" type="audio/mpeg">
Your browser support the audio element.
</audio>

Video Tag

The HTML <video> element embeds video content such as MP4, WebM, and Ogg files.

                    
<video controls>
<source src="video.mp4" type="video/mp4">
Your browser support the video element.
</video>

Image Tag

The HTML <img> element embeds an image into a document.

                                
 <img src="image.jpg" alt="Image Description">

10. Link

The HTML <a> element defines a hyperlink, which links the document to another location.

            
 <a href="https://www.example.com">Click here</a>

11. Form

The HTML <form> element defines an HTML form for user input.

                    
 <form>
Name: <input type="text" name="name">
<br>
Email: <input type="email" name="email">
<br>
<input type="submit" value="Submit">
</form>

Form Elements

HTML provides several form elements for user input, such as text fields, checkboxes, radio buttons, and submit buttons.

Text Field

The HTML <input> element with type="text" defines a single-line text input field.

        
 <input type="text" name="username" placeholder="Username">

Email Field

The HTML <input> element with type="email" defines a single-line text input field for email addresses.


<input type="email" name="email" placeholder="Email Address">

Password Field

The HTML <input> element with type="password" defines a single-line text input field for password entry.

                        
<input type="password" name="password" placeholder="Enter Password">

Checkbox

The HTML <input> element with type="checkbox" defines a checkbox input field and it's for multiple choice.

                        
 <input type="checkbox" name="terms" value="agree"> I agree to the terms<br>

Radio Button

The HTML <input> element with type="radio" defines a radio button input field and for single choice.

                                
<input type="radio" name="gender" value="male"> Male<br>
<input type="radio" name="gender" value="female"> Female<br>
<input type="radio" name="gender" value="other"> Other<br>

Submit Button

The HTML <input> element with type="submit" defines a submit button.

                                        
<input type="submit" value="Submit">

Selection List

The HTML <select> element defines a dropdown select list.

                                                
<select name="country">
<option value="usa">USA</option>
<option value="uk">UK</option>
<option value="canada">Canada</option>
</select>

TextArea

The HTML <textarea> element defines a multi-line text input field.

         
<textarea name="comments" rows="4" cols="50">Enter your comments here</textarea>

File Upload

The HTML <input> element with type="file" defines a file upload field.

        
<input type="file" name="file" accept="image/*">

Range Input

The HTML <input> element with type="range" defines a range input field.

        
<input type="range" name="volume" min="0" max="100" value="50">

Number Input

The HTML <input> element with type="number" defines a number input field.

        
<input type="number" name="age" min="18" max="99" value="30">

Color Picker

The HTML <input> element with type="color" defines a color picker field.

        
<input type="color" name="color" value="#ff0000">

Search Input

The HTML <input> element with type="search" defines a search input field.

        
<input type="search" name="search" placeholder="Search...">

Date Input

The HTML <input> element with type="date" defines a date input field.

        
<input type="date" name="dob" value="2022-01-01">

Time Input

The HTML <input> element with type="time" defines a time input field.

        
<input type="time" name="time" value="12:00:00">

CSS Integration

To style the HTML elements, you can use CSS. Here's an example:

        
<style>
    body { background-color: lightblue; }
</style>
<link rel="stylesheet" type="text/css" href="styles.css">

JavaScript Integration

To add interactivity to the form, you can use JavaScript. Here's an example:

                
<script>
    alert('Hello, World!');
</script>
<script src="script.js"></script>

Responsive Design

To make the form responsive, you can use CSS media queries. Here's an example:

        
<meta name="viewport" content="width=device-width, initial-scale=1">
<style>
@media (max-width: 600px) {
body { font-size: 18px; }
}
</style>
